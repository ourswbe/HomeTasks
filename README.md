# HomeTasks — система учёта домашних заданий

Веб-приложение для курсовой работы по теме домашних заданий.

## Возможности

- Регистрация и вход с выбором роли: учитель или ученик.
- При регистрации указывается класс (например, 8A, 9B).

- Учитель добавляет задания с предметом, дедлайном и типом:
  - обычное задание (текст / ссылка / фото);
  - тестовое задание (multiple choice).

- Для теста учитель задаёт количество вопросов и варианты ответов.
- Ученик выбирает один вариант ответа на каждый вопрос.

- После истечения дедлайна отправка блокируется.

- Учитель может:
  - принять работу как выполненную;
  - вернуть на доработку;
  - выставить оценку и комментарий.

- Ученик может:
  - отправить решение (текст, ссылка, фото);
  - пройти тест;
  - видеть статус проверки, оценку и комментарий.

- Отображаются дедлайны и статусы:
  **на проверке / выполнено**.

- Есть форма сброса пароля по email.

- Данные сохраняются в базе данных SQLite (по умолчанию) или PostgreSQL.

---

## Технологии

- Python 3.11
- Flask + Flask-SQLAlchemy
- SQLite или PostgreSQL
- Docker / Docker Compose
