# HomeTasks — система учёта домашних заданий

Веб-приложение для курсовой работы по теме домашних заданий.

## Возможности

- Регистрация и вход с выбором роли: **учитель** или **ученик**.
- Учитель добавляет задания с предметом, дедлайном и типом:
  - обычное задание (текст/ссылка/фото);
  - тестовое задание (**multiple choice**).
- Ученик видит список заданий, фильтрует по предмету и отправляет решение:
  - текст;
  - ссылка на файл/документ;
  - ссылка на фото;
  - выбранный вариант (для теста).
- Учитель видит отправки учеников и может:
  - принять как выполненное;
  - вернуть на доработку.
- Отображаются дедлайны и статусы: на проверке / выполнено.
- Данные сохраняются в базе данных SQLite.

## Технологии

- Python 3.11
- Flask
- Flask-SQLAlchemy
- SQLite
- Docker / Docker Compose

## Запуск локально

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Приложение будет доступно на: http://localhost:5000

## Запуск через Docker

```bash
docker compose up --build
```

После запуска откройте: http://localhost:5000

## База данных

База создаётся автоматически при первом запуске (`hometasks.db`).

## Основные экраны

- `/register` — регистрация (с выбором роли).
- `/login` — авторизация.
- `/tasks` — список заданий, фильтр по предмету, отправка решений и проверка.
- `/tasks/add` — добавление задания (только для учителя).
