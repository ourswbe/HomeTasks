# HomeTasks — система учёта домашних заданий

Веб-приложение для курсовой работы по теме домашних заданий.

## Возможности

- Регистрация и вход с выбором роли: **учитель** или **ученик**.
- Учитель может добавлять задания с предметом, описанием и дедлайном.
- Ученик видит список заданий и может отмечать их как выполненные.
- Фильтрация заданий по предмету.
- Отображение дедлайнов и статуса выполнения.
- Данные сохраняются в базе данных SQLite.

## Технологии

- Python 3.11
- Flask
- Flask-SQLAlchemy
- SQLite
- Docker / Docker Compose

## Запуск локально

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Приложение будет доступно на: http://localhost:5000

## Запуск через Docker

```bash
docker compose up --build
```

После запуска откройте: http://localhost:5000

## База данных

База создаётся автоматически при первом запуске (`hometasks.db`).

## Основные экраны

- `/register` — регистрация (с выбором роли).
- `/login` — авторизация.
- `/tasks` — список заданий, фильтр по предмету, статус выполнения.
- `/tasks/add` — добавление задания (только для учителя).
